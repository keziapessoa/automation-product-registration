# Product Registration Automation

Automates product registration in a web system using Python.

---

## 📌 Objective

Automate the manual process of product registration, reducing operational time and eliminating repetitive tasks.

---

## ⚙️ How It Works

The script performs the following steps:

1. Opens the browser
2. Accesses the web system
3. Logs in
4. Reads the product database (CSV)
5. Fills the form fields automatically
6. Repeats until all products are registered

**Workflow:**

Login → Read CSV → Loop through products → Completion

---

## 🛠 Tech Stack

| Technology | Purpose |
|------------|---------|
| Python 3   | Main programming language |
| PyAutoGUI  | GUI automation (mouse & keyboard) |
| Pandas     | CSV data handling |
